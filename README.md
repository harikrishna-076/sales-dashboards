DMart Retail Sales Performance Analysis | Power BI

Project Overview
This Power BI project focuses on analyzing the retail sales performance of DMart. The goal is to gain insights into sales, profit, loss, and quantity sold by cleaning raw Excel data and building interactive dashboards.

Key Objectives

Analyze overall sales and profit performance

Identify profit and loss trends

Track quantity sold across products and categories

Support business decision-making using data visualization

Data Cleaning & Preparation

Removed null values and missing records

Eliminated duplicate entries

Cleaned unwanted and inconsistent data

Performed data transformation using Power Query

Key Metrics Analyzed

Total Sales

Total Profit & Loss

Quantity Sold

Sales and profit trends
